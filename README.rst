atone
=====

Makes amends_

Features
--------

- create a new amend repository  (similar to ``git init``)
- clone an amend repository from amendhub (similar to ``git clone``)
- list amendments (similar to ``git log``)
- export amendments (similar to ``git format-patch``)
- add amendments  (similar to ``git am``)

.. _amends: https://amendhub.com/jcs/amend/
